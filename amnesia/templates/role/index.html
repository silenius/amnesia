Roles

List, add, and edit roles.

Roles allow you to fine tune the security and administration. A role defines a group of users that have certain privileges as defined in user permissions. Examples of roles include: anonymous user, authenticated user, moderator, administrator and so on. In this area you will define the role names of the various roles. To delete a role choose "edit".

Two user roles exist by default:

  • anonymous: this role is used for users that don't have a user account or that are not authenticated.
  • authenticated: this role is automatically granted to all logged in users.
